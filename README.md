# Учёт резцов (Flask + SQLite)

Минимальный прототип системы учёта резцов с реестром, карточкой резца, историей установок/переточек и экраном замены резцов.

## Запуск

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python app.py
```

Откройте `http://localhost:5000`.

## Быстрый старт с тестовыми данными

На странице «Реестр резцов» нажмите кнопку **«Создать тестовые данные»**.

## Основные маршруты

- `/cutters` — реестр резцов
- `/cutters/<id>` — карточка резца
- `/machines` — замена резцов

## Примечания

- База данных SQLite хранится в `instance/cutters.db`.
- Данные создаются автоматически при первом запуске.
